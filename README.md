# byd-vehicle-mock-server

A mock/emulator backend for `hass-byd-vehicle-plus`.

This project provides fake BYD vehicle data for:

- Home Assistant setup before vehicle delivery
- dashboard and automation preparation
- integration testing
- scenario simulation

## Goals

- Provide a configurable BYD-like backend
- Support multiple vehicle profiles and scenarios
- Allow users to simulate sensors, locks, climate, charging, GPS, and other vehicle features

## Project status

Early planning / bootstrap phase.
